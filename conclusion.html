Conclusion

We conclude this project by offering some insight into possible recommendations for the future. The first improvement regards including common knowledge announcements, which are currently not implemented in our program. Secondly, with our present program, only two agents (S and P) impact the outcome of the riddle. Therefore, another improvement would be adapting the model such that more than two agents can participate meaningfully, such as S and P. Finally, an excellent enhancement would be automatically transforming natural language to epistemic logic statements. In this way, any user could customize their riddle through different logic statements. This is not possible with the current implementation, as the user is supposed to be familiar with epistemic logic to access this feature of our program.
